Miss Universe Myanmar 2014 API

This is the unofficial API for Miss Universe Myanmar 2014 Competition written in Nodejs and Express framework.

Please note that this project is not affiliated with the actual organization. Source code is available at Github.

Base URL

http://missuniversemyanmar.herokuapp.com

GET /contestants

Getting all 20 contestants including their current votes and thumbnails.

GET /model/:name

Biography of individual contestant where :name is the name of the contestant in small case delimited by hyphen.

GET /model/eaint-myat-noe 
{
"status": "ok",
"biography": {
  "waistNumber": 20,
  "name": "Eaint Myat Noe",
  "age": 23,
  "citizenship": "Shan-Myanmar",
  "education": "B.C.Sc, Programming",
  "otherQualifications": "Winner of Focus Online Next Top  Model Season 2",
  "ambition": "To become successful women",
